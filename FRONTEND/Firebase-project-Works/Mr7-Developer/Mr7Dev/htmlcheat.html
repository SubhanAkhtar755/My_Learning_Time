HTML Cheat Sheet

Basic Tags

                    
 <html></html>: Creates an HTML document

                    
<head></head>: Sets off the title and other info that isn,t displayed

                    
<title></title>: Puts name of the document in the title bar

                    
<body></body>: Sets off the visible option of a document

Text Tags

                    
<pre></pre>: Creates performatted text

                        
<h1></h1>--><h6> </h6>: Creates headlines -- H1=Largest,H6=Smallest

                        
<b></b>: Creates bold text (should use <strong> instead)

                            
<i></i>: Creates italicized text (should use <em> instead)

                        
<tt></tt>: Creates typewriter-style text

                        
<code></code>: Use to define source code , usally monospace

                        
<cite></cite>: Creates a citation , usually processsd in italics

                        
<address></address>: Creates address section , usually processsd in italics

                        
<em></em>: Emphasizes a word (usually processsd in italics) 

                        
<strong></strong>: Emphasizes a word (usually processsd in bold)

                        
<font></font>: Sets size of font-1 to 7 (should use CSS instead)

Links


    
 <a href="URL"></a>: Creates a hyperlink to a Uniform Resource Locator

    
<a href="mailto:EMAIL_ADDRESS"></a>: Creates a hyperlink to an email address

    
<a name="Name"></a>: Creates a target location within a document

    
<a href="#Name"></a>: Creates a link to that target location

Formatting

                    
 <p></p>: Creates a new paragraph

                    
<br></br>: Alnserts a line break (carriage return)

                    
<blockquote></blockquote>: Puts content in a quote-indents text from both sides

                    
<div></div>: Used to format block content with CSS

                    
<span></span>: Used to format inline content with CSS

Forms

                    
 <form></form>: Defines a form

                    
<select multiple name="?" size="?"></select>: Creates a scrolling menu.Size sets the number of menu items visible before user needs to scroll.

                    
<select  name="?"></select>: Creates a pulldown menu sides

                    
<option></option>: Sets of each menu item

                    
<textarea name="?" cols="?" rows="?"></textarea>: Creates a textbox area.Columns set the width ; rows set the height 

                    
<input type="checkbox" name="?" value="?">:Creates a checkbox

                    
<input type="checkbox" name="?" value="? checked">:Creates a checkbox which is pre-checked 

                    
<input type="radio" name="?" value="?">:Creates a radio button

                    
<input type="radio" name="?" value="?" checked>:Creates a radio button which is pre-checked

                    
<input type="text" name="?" size="?">:Creates a one line text area .Size sets length,in characters

                    
<input type="submit" value="?">:Creates a submit button.Value sets the text in the submit button

                    
<input type="image" name="?" src="?" border="?" alt="?">:Creates a submit button using an image

                    
<input type="reset">:Creates a reset button

Lists

                    
 <ul></ul>: Creates an unordered list

                    
<ol start="?"></ol>: Creates an ordered list (start=xx,where xx is a counting number)

                    
<li></li>: Puts content in a quote-indents text from both sides

                    
<dl></dl>: Encompasses each list item 

                    
<dt></dt>: Creates a definition list

                    
<dd></dd>: Precedes each definition term

                    
Tables(use only for data layout)

                    
<table></table>: Creates a table 

                    
<tr></tr>: Sets of each row in a table

                    
<td></td>: Sets of each cell in a row

                    
<th></th>: Sets of the table header (a normal cell with bold , centerd text)

Input

                    
<input type="email" name="?">: Sets a single-line textbox for email addresses

                    
<input type="url" name="?">: Sets a single-line textbox for URLs

                    
<input type="number" name="?">: Sets a single-line textbox for a number

                    
<input type="range" name="?">: Sets a single-line textbox for a range of numbers

                    
<input type="date/month/week/time" name="?">: Sets a single-line textbox with a calender showing the date/month/week/time

                    
<input type="search" name="?">: Sets a single-line textbox for searching

                    
<input type="color" name="?">: Sets a single-line textbox for picking a color
